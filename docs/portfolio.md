## One-line description (for GitHub "About" / pinned repo)
RAG-backed Six Sigma Measure-phase coach that simulates a stakeholder and grades the
learner's analysis — grounded in cited methodology and statistics verified over MCP.

## LinkedIn post (2–3 sentences)
At value companion we're building an implementation-partner network with Lean and Six
Sigma consultancies — which got me curious: could an AI grade a process analysis the
way a good Black Belt does, on the evidence rather than vibes? So I built DMAIC Coach:
it guides a learner through the Measure phase, role-plays a sceptical stakeholder, and
then *evaluates the analysis* against a rubric — verifying every number itself over MCP
and citing the methodology behind each score. Runs in under five minutes with no API
key (deterministic mock provider), and the LLM is swappable (Claude/GPT/Grok).
